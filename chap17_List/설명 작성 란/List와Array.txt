#List
	데이터를 순서대로 저장하는 자료구조 중하나 
	여러 개의 요소를 포함할 수 있음
	배열(Array)과 달리 크기를 동적으로 조절할 수 있으며, 중복된 요소를 허용할 수 있다.
	util 패키지에 속해있음
	List 인터페이스 구현체는 ArrayList와 LinkedList
	
#List와 Array 차이점
	둘 다 데이터를 저장하는 자료구조
	Array : 배열을 생성할 때 크기를 지정, 크기를 변경할 수 없음
			배열의 크기를 변경할려면 새로운 배열을 생성하고 데이터를 복사해야됨
			특정 데이터 유형의 요소만 저장할 수 있음
			예를 들어 정수배열은 정수만 저장
	List : 크기가 가변적(바뀌거나 바꿀 수 있음)
			필요에 따라 요소를 추가하거나 제거할 수 있음
			크기 조정이 자동으로 처리
			여러 다른 유형의 요소를 저장할 수 있음 -> 혼합데이터 유형을 저장하는데 사용할 수 있음
