# Tree 트리
	계층적인 데이터 구조
	각 요소를 노드(Node)라고 부르며 이들 사이 관계는 가지(branch)로 표현
	계층 구조를 표현하고 데이터를 효율적으로 저장하고 검색하는 데 사용
	
# 주요 특징과 용어
	부모 노드 : 다른 하나 이상의 노드를 가지고 있는 노드
	자식 노드 : 부모 노드로부터 직접 연결된 노드, 자기 자신과 연결된 노드 중 자신보다 낮은 노드를 의미
	루트 노드 : 트리의 가장 상위에 있는 노드
			 모든 다른 노드들은 이 루트 노드에서 시작하는 경로를 통해 접근할 수 있음
			 뿌리 노드, 하나의 트리에서 하나밖에 존재하지 않음
			 부모 노드가 없음
	리트 노드 : 자식 노드가 없는 노드. 가장 최하위에 위치
			 단말 노드
	서브 노드 : 트리 내에서 특정 노드와 그 자손 노드들로 구성된 작은 트리
